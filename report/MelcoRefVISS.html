Usage cases

The server deployment is not bound to the infotainment system, i.e the actual IHU. The server can be deployed anywhere on any ECU inside the vehicle, but not only that the server could also be deployed outside the vehicle. The benefits of using internet based protocols is that the deployment of the signal server is boundless. It is a piece of software for the internet-of-things-everywhere.
